# Pengantar Machine Learning End-to-End

Selamat datang di tutorial singkat ini! Buku ini dirancang untuk memandu Anda melalui proses pembangunan model Machine Learning secara menyeluruh (*end-to-end*).

## Apa yang Akan Kita Pelajari?

Dalam perjalanan ini, kita tidak hanya akan fokus pada koding algoritma, tetapi juga tahapan penting lainnya:

1.  **Persiapan Data**: Membersihkan dan memformat data mentah.
2.  **Eksplorasi Data (EDA)**: Memahami pola dan karakteristik data.
3.  **Pemodelan**: Melatih model prediksi menggunakan Scikit-Learn.
4.  **Evaluasi**: Mengukur seberapa akurat model kita.
5.  **Deployment**: Konsep dasar membawa model ke produksi.

## Prasyarat

Sebaiknya Anda memiliki pemahaman dasar tentang Python. Kita akan banyak menggunakan pustaka seperti `pandas`, `numpy`, dan `scikit-learn`.
