# Tutorial Singkat End-to-End Machine Learning

Repositori ini berisi *source code* dan materi untuk buku **Tutorial Singkat End-to-End Machine Learning**.

## Deskripsi

Buku ini bertujuan untuk memberikan panduan praktis tentang siklus hidup proyek Machine Learning, meliputi:
- Persiapan dan pembersihan data
- Exploratory Data Analysis (EDA)
- Pemodelan (Modeling)
- Evaluasi
- Deployment sederhana

## Menjalankan Secara Lokal

Proyek ini dibangun menggunakan [MyST Markdown](https://mystmd.org/).

### Instalasi

Anda dapat menginstal MyST menggunakan `npm` atau `pip`:

```bash
# Menggunakan npm
npm install -g mystmd

# Atau menggunakan pip
pip install mystmd
```

### Membangun Buku

Untuk menjalankan server lokal dengan fitur *live reload* saat menulis:

```bash
myst start
```

Untuk membangun buku menjadi file HTML statis (biasanya di folder `_build`):

```bash
myst build
```
